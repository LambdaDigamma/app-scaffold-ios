# AppScaffold
> Provides the app chrome, coordinators and config.

## Installation

Add this project on your `Package.swift`

```swift
import PackageDescription

let package = Package(
    dependencies: [
        .package(url: "https://github.com/LambdaDigamma/app-scaffold-ios.git", majorVersion: 1, minor: 0)
    ]
)
```
